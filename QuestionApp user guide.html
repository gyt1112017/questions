1. Type 'cd code/QuestionApp/uczlfr1' (' ' is not included) in Bitvise xterm.

2. Type 'node httpServer.js &' and then 'phonegap serve' (' ' is not included).

3. If you use a computer, connect UCL VPN first (if you do not use UCL website) and then in any browser type the link: http://developer.cege.ucl.ac.uk:31270/ If you use an Android phone, connect UCL VPN through AnyConnect (if you do not use UCL website), open PhoneGap app, type 'developer.cege.ucl.ac.uk:31270' (' ' is not included) and press 'Connect' button.

4. Now you can see an interface. Click on the map, the latitude and longitude will be shown. Type them into relevant blank bars and write down your question, choices of answers and the correct answer.

5. When you are filling in all blank, just press 'Start Data Upload' button. If the data is successfully inserted into database, words at the bottom 'The result of the upload goes here' will change to 'Successfully uploaded'.

6. Check whether it is updated in database (The table is named PostGreSQL).
